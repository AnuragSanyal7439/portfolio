# Student Full Stack Developer Portfolio

A clean, responsive portfolio built with Next.js and Tailwind CSS for a student full stack developer.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Personalization

Update profile links, email, project URLs, and project details in `app/page.tsx`.

Replace `public/resume.pdf` with the final resume PDF before sharing or deploying.

## Deployment

This project is ready for Vercel:

1. Push the project to GitHub.
2. Import the repository in Vercel.
3. Use the default Next.js settings.
